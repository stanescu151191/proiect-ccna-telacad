Proiect Retea de Calculatoare - CCNA

Descriere Proiect

Am creat o infrastructura de retea complexa, distribuita pe 3 zone distincte, interconectate printr-o topologie redundanta si scalabila.

Reteaua 1 - HQ (SwitchBlock)

Sediul central foloseste o arhitectura de tip SwitchBlock cu doua switch-uri de distributie (sw-dist01 si sw-dist02) si trei switch-uri de acces (sw-acc01, sw-acc02, sw-acc03).

Caracteristici:

- EtherChannel LACP (Po1) intre sw-dist01 si sw-dist02

- HSRP cu load balancing pe 6 VLANuri

- STP pentru redundanta la nivel de acces

- DHCP pentru VLAN 50 (Printing)

VLANuri:

- VLAN 10 - Contabilitate : 10.10.10.0/24

- VLAN 20 - Vanzari : 10.10.20.0/24

- VLAN 40 - Productie : 10.10.40.0/24

- VLAN 50 - Printing : 10.10.50.0/24

- VLAN 77 - HR : 10.10.77.0/24

- VLAN 99 - Management : 10.10.99.0/24

Reteaua 2 - Productie (Router-on-a-Stick)

Sucursala foloseste o topologie Router-on-a-Stick prin R2, cu un switch de acces (sw-prod) care deserveste departamentele de Productie si Vanzari.

VLANuri:

- VLAN 20 - Vanzari : 10.20.20.0/24

- VLAN 40 - Productie : 10.20.40.0/24

- VLAN 50 - Printing : 10.20.50.0/24

- VLAN 99 - Management : 10.20.99.0/24

Reteaua 3 - Data Center

Serverele companiei sunt conectate direct la modulul de switch al routerului R3 si ofera servicii esentiale pentru intreaga companie.

Servicii disponibile:

- DNS : proiect-ccna.com

- HTTP : www.proiect-ccna.com

- MAIL : mail.proiect-ccna.com

- App : Aplicatie Contabilitate (10.30.10.10)

Serviciul de Email

Reteaua dispune de un server de email dedicat, accesibil din toate cele 3 retele. Fiecare statie de lucru are un cont de email dedicat, permitand comunicarea interna intre toate departamentele.

Server mail: mail.proiect-ccna.com (10.30.99.10)

Protocol intrare: POP3

Protocol iesire: SMTP

Conturi email create:

- conta@proiect-ccna.com : Departament Contabilitate

- vanzari@proiect-ccna.com : Departament Vanzari

- prod@proiect-ccna.com : Departament Productie

- hr@proiect-ccna.com : Departament HR

Exemplu comunicare:

Utilizatorul conta@proiect-ccna.com din Reteaua 1 poate trimite email catre prod@proiect-ccna.com din Reteaua 2, traficul traversand R1 si R2 in mod transparent pentru utilizator.

Disponibilitate:

- Accesibil din Reteaua 1 : 10.10.0.0/16

- Accesibil din Reteaua 2 : 10.20.0.0/16

- Accesibil din Reteaua 3 : 10.30.0.0/16

Interconectare

Cele 3 retele sunt interconectate prin routerul central R1 folosind rute statice cu sumarizare la nivel de /16.

- R1 - R2 : 172.16.0.8/30

- R1 - R3 : 172.16.0.16/30

- R1 - sw-dist01 : 172.16.0.0/30

- R1 - sw-dist02 : 172.16.0.4/30

- R2 - R3 : 172.16.0.12/30

IPv6

Reteaua suporta dual-stack IPv4 si IPv6:

- Net1 : 2001:ABC:1::/48

- Net2 : 2001:ABC:2::/48

- Net3 : 2001:ABC:3::/48

PC-urile primesc adrese IPv6 automat prin SLAAC.

Redundanta

- HSRP : failover gateway in sub 3 secunde

- STP : failover uplink in 30 secunde

- Po1 LACP : EtherChannel cu doua linkuri fizice

- Rutare redundanta : doua linkuri active simultan spre R1 cu distributia automata a traficului. Daca un link pica, traficul trece automat pe celalalt link fara intrerupere.
